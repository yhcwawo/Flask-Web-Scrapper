## Python Job Scrapper

`Python` + `Flask`를 활용하여 `replit` 환경에서 개발한 구인정보 탐색 Web Scrapper 입니다.

[Live Demo]("https://flask-web-scrapper.yhcwawo.repl.co/")

## Stack

`python` + `flask` + `replit`

## Result

다양한 인사정보 사이트들의 구인정보를 Scrapping하고 가져온 정보를 웹에서 확인하거나 csv파일로 추출 할 수 있습니다.
